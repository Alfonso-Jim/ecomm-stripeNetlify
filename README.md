- ES6
- Node.js

    - cookie-session v:1.4.0
    - ejs v:3.1.5
    - express v:4.17.1,
    - express-validator v:6.9.2
    - multer v:1.4.2
    - nodemon v:2.0.6
    - stripe v:8.132.0
- Bulma CSS
